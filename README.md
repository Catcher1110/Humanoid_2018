# Humanoid_2018

Prerequisites: 
- OpenGL
- Eigen
- CMake

How to compile:

1. Make a build folder
mkdir build

2. Go to the build folder and run cmake
cd build && cmake ..

3. compile
make

4. you now can see the three executable files:
simulation_mercury, simulation_valkyrie, simulation_atlas

run one of the simulations. for example:

./simulation_mercury
